# Calculadora de Média Escolar

Aplicação web que calcula a média de três notas e informa se o aluno foi aprovado ou reprovado.

## Tecnologias
- HTML
- CSS
- JavaScript

## Funcionalidades
- Entrada de três notas
- Cálculo automático da média
- Exibição do status:
  - Aprovado
  - Reprovado

## Como executar

Abra o projeto diretamente pelo navegador através do GitHub Pages:

https://barbaroto01.github.io/Calculo-media.Js/

## Aprendizados
Projeto criado para praticar:
- Operações matemáticas
- Manipulação de dados do usuário
- Lógica de decisão
